# Miele: The Best Cases

Miele Best Cases: This tool spread the Marketing knowledge along Miele Company. 

This is a Theme Tools that running in a standard Wordpress Platform

## Base on:

  - Wordpress
  - Stylus
  - jeet.gs
  - Bones Start Theme : https://github.com/eddiemachado/bones

## Concept

  - Home
    * Header
    * Bubble tags
    * Last Projects
    * Last Finding
  - Projects Page
    - Filter
    - Project Tiles
  - Project Details Page
    * Header
    * Topics
    * Outline
    * Findings list
  - Findigs Page
    - Filter
    - Finding Tiles
  - Finging Page
    - Header
    - Topics
    - Campaign Outline
    - Findings
    - Actions to be taken
